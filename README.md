### How to run on termux/vps: ###
- Copy paste the following cmds one by one
```
apt update
```
```
apt upgrade
```
```
pkg update && pkg upgrade
```
```
pkg install python -y
```
```
pkg install python2
```
```
pkg install git -y
```
```
git clone https://github.com/mouricedevs/calculator
```
```
cd calculator
```
```
python main.py
```
